Desarrollo de Aplicaciones Web - Ejercicios FP
¡Bienvenido al repositorio de ejercicios de las asignaturas del ciclo formativo de Desarrollo de Aplicaciones Web (DAW)! Aquí encontrarás el material y ejercicios prácticos organizados por asignaturas y módulos. Este repositorio está pensado para ayudar a reforzar los conocimientos adquiridos en clase y para facilitar el estudio autónomo.

📂 Estructura del Repositorio
El repositorio está organizado en carpetas correspondientes a cada asignatura o módulo. A continuación, se describen las principales carpetas y su contenido:

Programación: Ejercicios prácticos relacionados con la programación en lenguajes como Java y Python.
Desarrollo Web en Entorno Cliente: Ejercicios enfocados en tecnologías frontend, como:
HTML5
CSS3
JavaScript (ES6+)
Frameworks: Puede incluir librerías como Bootstrap o jQuery.
Desarrollo Web en Entorno Servidor: Ejercicios prácticos en backend usando:
PHP
Node.js
Bases de datos: Uso de MySQL o MongoDB.
Despliegue de Aplicaciones Web: Material sobre configuración de servidores, despliegue de aplicaciones, y herramientas como:
Docker
Apache
Nginx
Proyecto: Proyectos integrados o aplicaciones completas para poner en práctica los conocimientos adquiridos.

⚙️ Cómo usar este repositorio
Clonar el repositorio:

bash
Copy code
git clone https://github.com/usuario/repositorio-ejercicios.git
cd repositorio-ejercicios
Navegar por las carpetas para encontrar el contenido correspondiente a la asignatura que te interese.

Ejecutar ejercicios:

Si trabajas con PHP:
Usa un servidor local como XAMPP, MAMP, o Docker.
Para proyectos con Node.js:
Instala dependencias con npm install y ejecuta con node app.js.
Para bases de datos:
Importa el archivo .sql en tu gestor de bases de datos preferido.
Para ejercicios frontend:
Abre los archivos .html directamente en tu navegador o utiliza un servidor local como Live Server en VS Code.
